# 개요

웹브라우저를 통해 기자 출입처 실시간 오디오를 모니터링하고, 녹음된 파일들을 쉽게 접근할 수 있습니다.

## 시스템 구성 설명

- 주소: http://10.10.107.109
- 구성
  | 폴더 | 내용 |
  |--------------|-----------------------------------------------------------------------------|
  | help | 이 도움말 페이지
  | [휴지통](http://10.10.107.109/%ED%9C%B4%EC%A7%80%ED%86%B5/) | 24시간 지난 녹음파일 보관 |
  | [실시간모니터](http://10.10.107.109/%EC%8B%A4%EC%8B%9C%EA%B0%84%EB%AA%A8%EB%8B%88%ED%84%B0/) | 실시간으로 들어오는 출입처 직통 마이크 음성 모니터링(PC 모니터/이어폰 이용) |
  | [국방부외통부](http://10.10.107.109/%EA%B5%AD%EB%B0%A9%EB%B6%80%EC%99%B8%ED%86%B5%EB%B6%80/) | 최근 24시간 이내 녹음된 국방부, 외통부 직통 마이크 음성 파일 |
  | [고검국회](http://10.10.107.109/%EA%B3%A0%EA%B2%80%EA%B5%AD%ED%9A%8C/) | 최근 24시간 이내 녹음된 고검, 국회 직통 마이크 음성 파일 |

## 인제스트룸 업무 예시

1. 출입처 기자로부터 마이크 사용 요청 전화 접수
2. [실시간모니터](http://10.10.107.109/%EC%8B%A4%EC%8B%9C%EA%B0%84%EB%AA%A8%EB%8B%88%ED%84%B0/) 페이지에 접속하여 해당 출입처에서 오디오가 잘 들리는지 확인
3. 기자 마이크 사용
4. 해당 출입처 폴더에서 wav 파일 다운로드
   - 파일이 1개인 경우 그대로 사용
   - 2개로 나눠져 있을 경우 합쳐야 함
5. 다운로드한 wav파일을 가공하여 NDS에 등록

## 실시간모니터 페이지 설명

- 주소: http://10.10.107.109/실시간모니터/
- 사용법
  1. 모니터링할 출입처를 클릭하면 배경색이 파랗게 되며 해당 출입처 실시간 오디오 재생
  2. 모든 출입처 오디오를 동시에 들으려면 '전부 듣기' 버튼 클릭
  3. 전부 듣기 상태에서도 원하는 출입처 클릭하면 변경 가능
- 참고사항
  - `test`는 웹 페이지의 정상 작동을 테스트하기 위한 오디오임

## wav 파일 정보

- 16bit 48kHz 포맷
- 10분 단위로 분할, 분할된 파일들은 중단 없이 이어붙일 수 있음
- 녹음 완료된 10분짜리 파일의 이름에 오디오 레벨 값 표기(음수이므로 0에 가까울수록 큼)
- 녹음중인 파일은 파일명에 오디오 레벨 값 미표기
- 녹음중인 파일도 다운로드 가능
- 파일 클릭 시 나오는 화면에서 `보기` 선택 시 재생 가능(녹음중인 파일도 가능)

## 참고사항
